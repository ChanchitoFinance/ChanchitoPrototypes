# Documentation

This directory contains all documentation for the MVO Platform project, organized by dimension.

## Structure

### UX Documentation (`docs/UX/`)
- `design-principles.md` - Core UX principles and guidelines
- `color-system.md` - Color palette and usage guidelines
- `typography.md` - Typography system and rules
- `spacing.md` - Spacing system and usage
- `animations.md` - Animation guidelines and examples

### Architecture Documentation (`docs/architecture/`)
- `overview.md` - Project overview and tech stack
- `authentication.md` - Authentication setup and usage
- `payments.md` - Payment integration guide

## Quick Start

1. Read `architecture/overview.md` for project structure
2. Review `UX/design-principles.md` for design guidelines
3. Check `architecture/authentication.md` for auth setup
4. See `architecture/payments.md` for payment integration

## Contributing

When adding new features:
1. Update relevant documentation files
2. Add examples where applicable
3. Keep documentation in sync with code changes
